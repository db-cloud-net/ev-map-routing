## Valhalla (aligned)

- **Canonical doc:** [`VALHALLA.md`](./VALHALLA.md) — port is almost always **8002** (not `:800`).
- **Code:** `api/src/config/valhallaBaseUrl.ts` — `DEFAULT_VALHALLA_BASE_URL=http://valhalla:8002`, `getValhallaBaseUrl()` used by `valhallaClient.ts`.
- **Your NAS:** reachable at `http://192.168.86.38:8002` (e.g. `GET /status` works). Set **`VALHALLA_BASE_URL`** in **`.env`** to that origin when the API runs on your PC.
- **Docker:** mirror compose was removed; set **`VALHALLA_BASE_URL`** explicitly for planner containers.

## API / web ports

- **API** defaults to **3001** (`api/src/server.ts`); **web** **3000**. Avoid **EADDRINUSE** — one process per port; kill stale Node or use another port.
- **`npm run dev:api`** / **`npm run dev:web`** from repo root (`Travel-Routing`), not `C:\Users\david` alone.
